feat(client-portal): Implement client portal integration - Stage 1

This commit implements the first stage of the client portal integration for the VARAi Commerce Studio platform. It includes:

- Core UI components (Dashboard, Onboarding Wizard, Report Viewer, Settings, Plugin Management)
- API integration with the client portal
- Performance optimizations (caching, memoization, debouncing)
- Comprehensive tests (integration, API, responsive design)

The implementation enables key user journeys:
- Client registration and onboarding
- Dashboard access for metrics and reports
- Plugin management and configuration
- Settings management
