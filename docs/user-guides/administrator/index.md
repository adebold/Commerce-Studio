# VARAi Administrator Guides

Welcome to the VARAi Administrator Guides. These guides are designed to help system administrators effectively manage the VARAi platform infrastructure, user access, and security.

## Available Guides

### [System Administration Guide](./system-administration.md)
Learn how to set up, configure, and maintain the VARAi platform infrastructure.

### [User Management Guide](./user-management.md)
Understand how to create, edit, and manage user accounts in the VARAi platform.

### [Role and Permission Management Guide](./role-permission-management.md)
Learn how to set up and manage roles and permissions for different user types in the VARAi platform.

### [Tenant Management Guide](./tenant-management.md)
Understand how to manage multiple tenants (organizations) in the VARAi platform.

### [System Monitoring Guide](./system-monitoring.md)
Learn how to monitor the performance, health, and usage of the VARAi platform.

### [Security Best Practices](./security-best-practices.md)
Understand the security features of the VARAi platform and learn best practices for securing your implementation.

## Getting Started

If you're new to administering the VARAi platform, we recommend starting with the [System Administration Guide](./system-administration.md) to understand the overall architecture and configuration options.

## Need Help?

If you can't find the information you need in these guides, please contact our technical support team at admin-support@varai.ai.
