VARAi Commerce Studio API Documentation

Welcome to the VARAi Commerce Studio API. Our REST API allows you to integrate directly with our platform to manage stores, products, orders, and analytics programmatically.

Base URL

https://api.varai.ai/v1

Quick Start

Get started with our API in minutes:

  1. Sign up for a VARAi Commerce Studio account
  2. Generate your API key from the dashboard
  3. Make your first API call

Authentication

The VARAi API uses API key authentication. Include your API key in the Authorization header of all requests.

GET /auth/verify

Verify API Key

Test your API key and get account information.

Headers
Parameter Type Required Description
Authorization string Required Bearer {your_api_key}
Example Request
curl -X GET "https://api.varai.ai/v1/auth/verify" \ -H "Authorization: Bearer your_api_key_here"
Example Response
{ "success": true, "account": { "id": "acc_123456789", "name": "Your Company", "plan": "professional", "api_version": "v1" } }

Store Integrations

Connect and manage e-commerce platform integrations including Shopify, Magento, BigCommerce, WooCommerce, and VisionCraft stores.

POST /stores/connect

Connect Store

Connect a new e-commerce store to your VARAi account.

Request Body
Parameter Type Required Description
platform string Required Platform type: shopify, magento, bigcommerce, woocommerce, visioncraft
store_url string Required Store URL (e.g., mystore.myshopify.com)
api_key string Optional API key for platform authentication
access_token string Optional OAuth access token for supported platforms
store_name string Optional Display name for the store
Example Request
curl -X POST "https://api.varai.ai/v1/stores/connect" \ -H "Authorization: Bearer your_api_key_here" \ -H "Content-Type: application/json" \ -d '{ "platform": "shopify", "store_url": "mystore.myshopify.com", "access_token": "shpat_xxxxxxxxxxxxx", "store_name": "My Awesome Store" }'
Example Response
{ "success": true, "platform": "shopify", "store_id": "store_123456789", "store_name": "My Awesome Store", "message": "Store connected successfully", "integration_status": "active" }
GET /stores

List Connected Stores

Retrieve all stores connected to your account.

Example Response
{ "success": true, "stores": [ { "store_id": "store_123456789", "platform": "shopify", "store_name": "My Awesome Store", "store_url": "mystore.myshopify.com", "status": "active", "connected_at": "2025-06-23T10:30:00Z", "last_sync": "2025-06-23T10:45:00Z" } ], "total": 1 }

Product Management

Manage products across all connected stores with AI-powered recommendations and optimization.

GET /products

List Products

Retrieve products from connected stores with filtering and pagination.

Query Parameters
Parameter Type Required Description
store_id string Optional Filter by specific store
category string Optional Filter by product category
limit integer Optional Number of products to return (default: 50, max: 200)
offset integer Optional Number of products to skip (default: 0)
Example Response
{ "success": true, "products": [ { "product_id": "prod_123456789", "store_id": "store_123456789", "title": "Premium Wireless Headphones", "description": "High-quality wireless headphones with noise cancellation", "price": 299.99, "currency": "USD", "sku": "WH-1000XM4", "category": "Electronics", "images": [ "https://cdn.example.com/product1.jpg" ], "inventory": 45, "status": "active", "ai_recommendations": { "price_optimization": "Consider reducing price by 5% to increase sales", "seo_score": 85, "conversion_potential": "high" } } ], "pagination": { "total": 150, "limit": 50, "offset": 0, "has_more": true } }
POST /products/optimize

AI Product Optimization

Get AI-powered recommendations for product optimization including pricing, descriptions, and SEO.

Request Body
Parameter Type Required Description
product_id string Required Product ID to optimize
optimization_type array Optional Types: ["pricing", "description", "seo", "images"]
Example Response
{ "success": true, "product_id": "prod_123456789", "optimizations": { "pricing": { "current_price": 299.99, "recommended_price": 284.99, "confidence": 0.87, "reasoning": "Market analysis suggests 5% price reduction could increase sales by 15%" }, "description": { "current_length": 45, "recommended_improvements": [ "Add more technical specifications", "Include use case scenarios", "Highlight unique selling points" ] }, "seo": { "current_score": 85, "recommended_keywords": ["wireless headphones", "noise cancelling", "premium audio"], "title_optimization": "Premium Wireless Noise-Cancelling Headphones - Studio Quality" } } }

Webhooks

Set up webhooks to receive real-time notifications about events in your stores.

POST /webhooks

Create Webhook

Register a webhook endpoint to receive event notifications.

Request Body
Parameter Type Required Description
url string Required Webhook endpoint URL
events array Required Events to subscribe to: order.created, order.updated, product.updated
secret string Optional Secret for webhook signature verification
Available Events
  • order.created - New order placed
  • order.updated - Order status changed
  • product.created - New product added
  • product.updated - Product information changed
  • store.connected - New store connected
  • store.disconnected - Store disconnected

Rate Limits

The VARAi API implements rate limiting to ensure fair usage and optimal performance.

Rate Limit Tiers

Plan Requests per minute Requests per hour Burst limit
Starter 60 1,000 100
Professional 300 10,000 500
Enterprise 1,000 50,000 2,000

Rate Limit Headers

All API responses include rate limit information in the headers:

X-RateLimit-Limit: 300 X-RateLimit-Remaining: 299 X-RateLimit-Reset: 1640995200

Error Handling

The VARAi API uses conventional HTTP response codes and returns detailed error information in JSON format.

HTTP Status Codes

Code Description
200 OK - Request successful
201 Created - Resource created successfully
400 Bad Request - Invalid request parameters
401 Unauthorized - Invalid or missing API key
403 Forbidden - Insufficient permissions
404 Not Found - Resource not found
429 Too Many Requests - Rate limit exceeded
500 Internal Server Error - Server error

Error Response Format

{ "success": false, "error": { "code": "INVALID_API_KEY", "message": "The provided API key is invalid or has expired", "details": { "field": "authorization", "suggestion": "Please check your API key and ensure it's correctly formatted" } }, "request_id": "req_123456789" }

SDKs & Libraries

Official SDKs and libraries to integrate with the VARAi API in your preferred programming language.

Official SDKs

JavaScript/Node.js

npm install @varai/commerce-studio
const { VaraiClient } = require('@varai/commerce-studio'); const client = new VaraiClient({ apiKey: 'your_api_key_here' }); // Connect a store const store = await client.stores.connect({ platform: 'shopify', store_url: 'mystore.myshopify.com', access_token: 'shpat_xxxxx' });

Python

pip install varai-commerce-studio
from varai import VaraiClient client = VaraiClient(api_key='your_api_key_here') # Connect a store store = client.stores.connect( platform='shopify', store_url='mystore.myshopify.com', access_token='shpat_xxxxx' )

PHP

composer require varai/commerce-studio
use Varai\CommerceStudio\Client; $client = new Client('your_api_key_here'); // Connect a store $store = $client->stores()->connect([ 'platform' => 'shopify', 'store_url' => 'mystore.myshopify.com', 'access_token' => 'shpat_xxxxx' ]);

Ruby

gem install varai-commerce-studio
require 'varai/commerce_studio' client = Varai::CommerceStudio::Client.new( api_key: 'your_api_key_here' ) # Connect a store store = client.stores.connect( platform: 'shopify', store_url: 'mystore.myshopify.com', access_token: 'shpat_xxxxx' )
